## Theme
With the help of this tool you can change your termux shell to look like Boss.

## How to Install Themes In Termux.

`$ apt update -y`

`$ apt upgrade -y`

`$ apt install figlet -y`

`$ apt install ruby -y`

`$ gem install lolcat -y`

`$ pkg up -y`

`$ pkg install git -y`

`$ git clone https://github.com/Boss-BD/theme`

`$ cd theme.sh`

`$ chmod +x *`

`$ bash theme.sh`

`$ Yes`
Now Restart Termux.
